# Enscript-Carving

    Encase에서 지원하는 Enscript를 활용한 파일 카빙 스크립트입니다.

    "Unallocated Clusters" 영역에서 MS-Office의 문서인 "Docx"와 스프레드시트 "Xlsx"를 찾아 Carving을 수행합니다.  

    원본 확장자는 ".enscript" 입니다.

    Encase v8.08 환경에서 테스트 완료.
